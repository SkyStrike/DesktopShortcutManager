# Desktop Shortcut Manager (Retired)

**Notice: This project is officially retired and has been superseded by the new [Shortcut Manager](https://github.com/SkyStrike/shortcut-manager), which is built using WinUI 3.**

---

A small program for managing desktop shortcuts, specifically designed for users who prefer a clean desktop environment.

## History

This application was originally written in .NET 2.0 in 2011. After a long period without updates, it was ported to .NET 4.8 to include several features and improvements that were previously missing.

The port from .NET 2.0 to 4.8 involved code reorganization and cleanup, but the core functionality remains consistent with the original vision.

## Features

- **Shortcut Management:** Organize your desktop shortcuts into groups.
- **Icon Extraction:** Automatically extracts icons from executables and shortcuts.
- **Custom Sorting:** Sort shortcuts within groups.
- **Patcher Utility:** Includes tools for rebuilding icons, clearing unused icons, and fixing application paths.

## Technical Details

- **Framework:** .NET 4.8
- **Language:** C#
- **UI:** Windows Forms
- **Data Storage:** XML-based configuration files for shortcuts and groups.

## Components

- **Modules:** Core logic for shortcut operations, configuration management, and icon extraction.
- **Entities:** Data models for Shortcuts, Shortcut Groups, and Icon Mappings.
- **Forms:** User interface for managing shortcuts, groups, and application settings.

## Getting Started

1. Clone the repository.
2. Open the solution in Visual Studio.
3. Build and run the project.
